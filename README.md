# Multi-Agent AI System for Autonomous Game Development

Production-ready multi-agent pipeline using LangGraph + Meta Llama 3.3 70B on Databricks 
that autonomously generates, tests, and refines Python code — scoring 9.5/10 on first iteration.

## Key Features
- 7-agent orchestration (Director → Architect → Engineer → QA → Scorer)
- Nested QA Subgraph with AST syntax validation + LLM logic testing
- Human-in-the-Loop checkpoints
- 9.5/10 quality score, 100% logic test pass rate, zero syntax errors

## Tech Stack
LangGraph · LangChain · Databricks Mosaic AI · Meta Llama 3.3 70B · Python · Pygame
